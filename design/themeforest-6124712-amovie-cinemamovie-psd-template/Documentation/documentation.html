Each psd file contacins 3 main groups of layers: HEADER, BODYCOPY, FOOTER, which are named accordingly to page structure. So it's easy to navigate through psd files.

If you want to change colors you can do this easily almost for every element (there are used shapes in psd-files to make it easy to customize template). You need just double click on layer thumbnail with shape in layers panel. Color pannel will be opened and you may choose color you need.

Icons can be easily edit. You can change size and color in Photoshop Character pannel if FontAwesome font is installed on your device (or even while coding).

Photos aren't included in this template.
